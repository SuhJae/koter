# 코터 (koter)

**과몰입 한국어 터미널.** 리눅스 셸의 명령어, 하위 명령어, 옵션까지 전부 한국어로 강제하는 bash/zsh 확장입니다. 음역(node → 노드)은 인정하지 않으며, 오직 진심의 번역만 수록합니다. 현재 번역 391개.

```console
$ 뭐 --길게
total 8
-rw-r--r-- 1 suhjae 4096  7월  2 메인.go

$ 멍청이 저질러 -메시지 "버그 수정"     # git commit -m "버그 수정"
$ 멍청이 밀어                          # git push
$ 멍청이 역사왜곡 main                  # git rebase main
$ 어명 장터 설치 이쁜꼭대기              # sudo apt install htop
$ 뱀 안녕.py                           # python3 안녕.py
$ 탈출게임 설정.txt                     # vim (나올 수 없기 때문)
$ 황회장님                             # nvidia-smi
$ 환생                                 # reboot
```

영어를 입력하면 실행되지 않습니다.

```console
$ ls
세종대왕님이 보고 계십니다. '뭐'라고 하세요.
```

인자 위치도 검사하므로 우회할 수 없습니다.

```console
$ 어명 apt install htop
세종대왕님이 보고 계십니다. 'apt' 말고 '장터'라고 하세요.
$ 어명 장터 install htop
세종대왕님이 보고 계십니다. 'install' 말고 '설치'라고 하세요.
$ 어명 장터 설치 htop
세종대왕님이 보고 계십니다. 'htop' 말고 '이쁜꼭대기'라고 하세요.
$ 어명 장터 설치 이쁜꼭대기
[sudo] password for suhjae:          # 이제야 통과됩니다
$ 뭐 -l
세종대왕님이 보고 계십니다. '-l' 말고 '--길게'라고 하세요.
```

`cd`는 `이동`, `가즈아`, `가` 등 동의어를 지원합니다.

## 맛보기

설치 없이 현재 터미널에서만 체험합니다. 탭을 닫으면 흔적 없이 사라집니다.

```sh
source <(curl -fsSL https://raw.githubusercontent.com/suhjae/koter/main/코터.sh)
```

## 설치

```sh
curl -fsSL https://raw.githubusercontent.com/suhjae/koter/main/install.sh | sh
```

`~/.코터/`에 파일 하나를 내려받고 `.bashrc`/`.zshrc`에 source 한 줄을 추가합니다. 설정 파일은 만들지 않습니다.

## 사용법

| 명령 | 동작 |
|---|---|
| `코터` | 전체 사전 출력 |
| `영어로` | 현재 세션만 영어로 복귀 (즉시) |
| `한국어로` | 다시 한국어로 |
| `코터그만` | 완전 삭제 — rc 파일의 한 줄과 `~/.코터/` 제거 |

## 대표 번역

| 한국어 | 영어 | 근거 |
|---|---|---|
| `멍청이` | git | 리누스 토르발스가 명명한 원뜻 그대로 |
| `탈출게임` | vim | 설명 생략 |
| `이양고` | tac | cat의 역순이므로 고양이의 역순 |
| `주전자` | http | RFC 2324 (418 I'm a teapot) |
| `쉿` | ssh | 발음 |
| `어명` | sudo | 거역할 수 없음 |
| `흑역사` | history | 사실관계 부합 |
| `의절` | disown | 직역 |
| `호적파기` | userdel | 직역에 가까움 |
| `천지창조` | terraform | 직역 |
| `반갈죽` | git bisect | 이진 탐색으로 절반씩 가름 |

전체 목록은 터미널에서 `코터`를 실행하면 확인할 수 있습니다.

## 안전 설계

농담이지만 셸은 소중하므로, 다음을 보장합니다.

- **대화형 셸에서만 동작합니다.** 스크립트, cron, CI 환경은 어떤 영향도 받지 않습니다.
- **직접 입력한 명령만 거절합니다.** 파이프 내부, 프롬프트(PS1), 자동완성, `source`된 스크립트에서 호출된 영어 명령은 조용히 통과시킵니다. 셸이 죽지 않아야 농담이 유지됩니다.
- **기존 함수를 보존합니다.** `.bashrc`의 `cd` 래퍼 등 기존에 정의된 함수는 저장해 두었다가 `영어로` 실행 시 그대로 복원합니다.
- **실제 파일명은 번역하지 않습니다.** `설치`라는 이름의 파일이 있으면 그대로 전달됩니다.
- 문제가 생기면 `영어로` 한 번으로 즉시 원상복구됩니다.

## 구조 및 기여

```
사전/명령.txt   명령어 사전        (형식: 영어 한국어1 [한국어2 ...])
사전/단어.txt   하위명령어·옵션 사전
엔진.sh         런타임 템플릿
조립.sh         사전 + 엔진 → 코터.sh 생성기
코터.sh         생성물 (curl 한 줄 설치를 위해 커밋함)
test.sh         자가 진단 (20개 검사)
```

`코터.sh`는 직접 수정하지 않습니다. 사전이나 엔진을 고친 뒤 `./조립.sh`로 재생성합니다.

### 새 명령어 추가

`사전/명령.txt`에 한 줄 추가합니다. 형식은 `영어명령 한국어1 [한국어2 ...]`입니다.

```sh
echo "kubectl 조타수" >> 사전/명령.txt
./조립.sh && bash test.sh
```

하위 명령어나 옵션은 `사전/단어.txt`에 추가합니다. 여기 등록된 단어는 모든 명령의 인자 위치에서 번역·검사됩니다.

```
compose 작곡          # 부두꾼 작곡 → docker compose
--follow --졸졸       # 일기장 --졸졸 → journalctl --follow
```

### 동의어(별칭) 추가

같은 줄 뒤에 이어 씁니다. 순서가 곧 서열입니다 — **첫 번째 한국어가 대표어**로, 영어를 입력했을 때 거절 메시지에 표시되는 단어입니다.

```
cd 이동 가즈아 가      # 셋 다 동작, 혼날 때는 "'이동'라고 하세요"
```

대표어를 바꾸려면 순서만 바꾸면 됩니다.

### 번역 교정 (더 나은 번역 제안)

기존 줄의 한국어를 고치면 됩니다. 단, 이미 쓰이던 단어를 지우면 그 단어로 익힌 사용자의 손이 헛돌게 되므로, 확신이 없다면 새 번역을 대표어 자리에 넣고 기존 단어를 동의어로 남기는 방식을 권합니다.

```
# 변경 전
top 꼭대기
# 변경 후 — 새 대표어 + 기존 단어 유지
top 정상 꼭대기
```

### 규칙

- **한국어 중복은 빌드가 거부합니다.** 같은 한국어가 두 영어를 가리키면 `조립.sh`가 목록을 출력하고 실패하므로, 충돌 걱정 없이 추가해도 됩니다.
- `#`으로 시작하는 줄은 주석입니다.
- 기여 기준: 음역 금지, 직역·의역·말장난 환영, 웃기면 병합.
